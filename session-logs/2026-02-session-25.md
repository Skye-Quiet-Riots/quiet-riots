## Key Decisions

- **No guest mode** — User explicitly requested removing "continue as guest". All interactive components wrapped with AuthGate (block mode). APIs already return 401 as second layer.
- **Cookie consent** — PostHog SDK doesn't load until analytics consent given. Consent stored in cookie + DB.
- **AuthGate a11y** — Wrap mode now handles keyboard (Enter/Space opens modal, Escape closes). Focus trapped in modal.
- **`'\***'`as global sentinel** — Legal documents use`'\*\*\*'`(not`'global'`) for country_code fallback because the column has CHECK(length <= 3).
- **Flaky test fix** — `bot-api.test.ts` "tracks issue_id from params" was flaky due to timestamp ties across events. Fixed with `find()` instead of `events[0]`.

## Discoveries

- `Intl.ResolvedNumberFormatOptions.minimumFractionDigits` can be `undefined` in TypeScript — needs `?? 2`
- React ESLint rule `react-hooks/set-state-in-effect` blocks `setState` in `useEffect` — use lazy state initializer instead
- SQLite `ALTER TABLE ADD COLUMN` has no `IF NOT EXISTS` — migration tests must drop ALL tables between runs (dynamic approach via `sqlite_master`)
- Nav bar tests break when Profile is moved to avatar dropdown — need to test for `aria-label` instead
