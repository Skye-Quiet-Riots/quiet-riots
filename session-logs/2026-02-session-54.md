## Decisions

- **Mandatory PLAN.md rule:** Added to CLAUDE.md — any multi-step task (3+ steps) MUST have its plan written to PLAN.md, committed, and pushed BEFORE implementation code. This prevents the kind of lost work that triggered this session.
- **Script-first translation protocol:** Replaced the sub-agent approach (6 parallel Task agents) with a mandatory Node.js script approach in CLAUDE.md. Sub-agents were hitting "Prompt is too long" errors on large locale files (fr, nl, es, etc.). The script approach is faster, more reliable, and doesn't hit token limits.
- **Migration uses LIKE matching:** Production has more data than seed, so migration matches descriptions by prefix (LIKE) rather than by ID.
- **Stale translations deleted:** Migration deletes old translations for updated descriptions so they fall back to corrected English until regenerated.
