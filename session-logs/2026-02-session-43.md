## Decisions & Rationale

- **Phone auth is dev-only for now:** Codes are logged to console in dev mode. Production SMS integration (Twilio/etc.) deferred until needed — the backend is ready.
- **Anti-enumeration:** Phone send-code always returns success even for unknown phones (same pattern as magic link emails)
- **Brand terms kept in English in translations:** "Quiet Riot", "Quiet Rioter", "First Quiet Rioter", "Setup Guide", "WhatsApp" — never translated
- **Incremental commits:** Each phase committed separately for clean git history and rollback ability

## Gotchas Discovered

- **Translation agents hallucinate extra keys:** The Admin fix agent added 14 keys that don't exist in en.json (activeCampaigns, activeToday, campaigns, etc.) to 6 locales. Caught by the `messages.test.ts` key parity check. Always verify translations against the English source.
- **Agent context overflow on large locales:** Locale files are ~600 lines each. Agents processing >10 locales at once can exhaust context. Split into batches of 5-6 for reliable completion.
