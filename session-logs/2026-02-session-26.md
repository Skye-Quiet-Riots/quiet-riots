## Key Decisions

- **No API key needed:** Translations generated directly by Claude sub-agents rather than requiring Google Translate API key. The `generate-locale-messages.ts` script still supports API-based translation for future use.
- **Parallel translation:** Used 16 concurrent sub-agents to translate all 44 locales in ~10 minutes, grouped by language family
- **Brand names stay English:** "Quiet Riot", "Quiet Riots", "Quiet Rioters", and "OpenClaw" remain in English in all locales
- **"rioters" localized:** Translated to local equivalents — participantes (es), participants (fr), Teilnehmer (de), 参加者 (ja), 참여자 (ko), участники (ru), etc.

## Discoveries

- Russian ICU plurals need `few`/`many` forms in addition to `one`/`other` — the agents correctly added these
- CJK languages (zh-CN, zh-TW, ja, ko) don't have grammatical plural — use same word for both one/other forms
- File validation with `it.each()` in Vitest works well for testing all 44 locales without code duplication
