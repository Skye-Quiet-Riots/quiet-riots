## Summary

User compared the live site (quietriots.com) with a mockup and identified several gaps:
1. Mobile: Join/Follow buttons buried at bottom of page, invisible without scrolling
2. JoinButton uses wrong colour (dark instead of blue)
3. FollowButton is undersized
4. Homepage hero is visually heavier than the mockup's cleaner design

A comprehensive 3-phase plan was created and reviewed through the mandatory 3-round, 7-reviewer protocol. The plan was approved by the user.

## What Was Done

### CLAUDE.md Enforcement
Added 3 enforcement layers to prevent skipping the review protocol:
1. New gotcha in Critical Gotchas: "ExitPlanMode requires completed reviews"
2. New bullet in incremental commit protocol: "Complete 3-round reviews before seeking approval"

### 3-Round Plan Review
- **Round 1:** 7 reviewers found ~20 CRITICAL/HIGH issues — major additions: dark mode on CTA bar, blue-700 for contrast (6.1:1), feed_comments table schema, full API contracts, bot actions, photo upload security, ICU pluralisation, 13 i18n keys
- **Round 2:** 7 reviewers found 3 CRITICAL issues:
  - `force-dynamic` MISSING from issue detail page (Round 1 incorrectly claimed present)
  - MobileCTABar + MobileBottomNav z-40 overlap
  - No test for force-dynamic export
  - Also: descoped homepage feed filter tabs (homepage has no community feed)
- **Round 3:** All 7 specialists APPROVED, zero CRITICAL/HIGH findings

### Key Design Decisions
- **3-breakpoint strategy:** `<sm` (inline CTA), `sm-lg` (fixed bottom bar), `lg+` (sidebar)
- **blue-700** for JoinButton (6.1:1 contrast vs white, exceeds WCAG AA 4.5:1)
- **Phase 3g descoped:** homepage has PersonalFeed not community feed
- **13 i18n keys** including ICU pluralisation for comment counts

## Bug Discovered
- Issue detail page (`issues/[id]/page.tsx`) is MISSING `export const dynamic = 'force-dynamic'` — has JoinButton, FollowButton, FeedSection, etc. (all client components) but no force-dynamic. CSP blocks their JavaScript in production. This is a pre-existing bug, not caused by this redesign.
