## Key Decisions

1. **UNION ALL across 3 tables** — Personal feed aggregates feed posts, evidence submissions, and approved/featured riot reels from issues the user has joined or followed
2. **Compound cursor** — `{iso_datetime}_{uuid}` format with `lastIndexOf('_')` to handle hyphens in UUIDs
3. **5-page cap** — Client-side limit to prevent excessive loading; shows "Load more" button
4. **Private cache** — `Cache-Control: private, no-store` since feed is user-specific
5. **LEFT JOIN for riot reels** — `submitted_by` is nullable for curated reels; falls back to "Quiet Riots" name
