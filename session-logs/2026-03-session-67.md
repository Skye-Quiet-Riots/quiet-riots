## Key Decisions

1. **Wallet integration** — chickens are paid from the user's existing riot wallet (same as action initiatives)
2. **Status workflow** — paid → accepted → in_progress → delivered (with cancel/refund/dispute branches)
3. **Express delivery** — optional surcharge for 2-day vs 5-day standard
4. **Country-based pricing** — different base prices per country with currency support
5. **Issue/org linking** — optional linking of chicken deployments to issues/organisations for themed protests

## Surprising Findings

- Haiku's 32K output token limit means ~101 keys × 55 locales must be split into 6+ batches, not 1 or 3
- The extraction script needed brace-matching JSON parser + per-locale extraction fallback to handle truncated agent outputs
- bg-Latn was worst for Cyrillic contamination (26 strings) — the model uses Bulgarian characters like `то` and `та` as suffixes
