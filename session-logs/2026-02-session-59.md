## Key Decisions

| Decision | Rationale |
|----------|-----------|
| `Intl.DisplayNames` for country names | Zero DB rows, zero API cost, built into Node.js |
| Romanised locales fall back to English for countries | `Intl.DisplayNames` returns native script for -Latn locales |
| Actions keyed by title (not ID) in translation files | Titles are unique across the 39 action types; IDs vary per issue |
| Expert `name` stays untranslated | Proper nouns (Dr. Patel, Prof. Johnson) |
| Only curated reel titles translated | Community-submitted reels keep YouTube titles |
| Auto-merge when CI green | User requested — removes bottleneck |
