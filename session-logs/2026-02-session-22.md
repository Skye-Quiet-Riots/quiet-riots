### Key decisions

- **nohup with sleep for worktree cleanup** — the fire-and-forget approach lets the session exit before the directory is deleted. Simpler and safer than printing manual cleanup commands for the user.
- **Auto-merge as default** — PRs should be merged in the same session they're created. The `--admin` flag bypasses the 1-approval branch protection. Only leave PRs open if the user explicitly requests review.

### Discoveries

- **Worktree removal is fatal mid-session** — deleting the worktree directory while Claude Code is running permanently breaks all Bash commands. The persisted cwd no longer exists and there's no recovery mechanism.
- **`gh pr merge` fails in worktrees** — `fatal: 'main' is already used by worktree`. Workaround: `gh api repos/.../pulls/.../merge -X PUT -f merge_method=squash`.
